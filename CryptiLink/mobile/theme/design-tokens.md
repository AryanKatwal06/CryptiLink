# Design Tokens

Dark-first design tokens. Use these primitives; do not hardcode values in components.

Typography

- Font family: Inter
- Sizes: 12, 14, 16, 20, 24

Spacing

- 4, 8, 12, 16, 24, 32, 48, 64

Motion

- 150ms, 250ms, 400ms

Colors

- Background, surface, text, muted

Usage

- Import tokens from `mobile/theme/tokens.ts` and consume in styles/themes.
