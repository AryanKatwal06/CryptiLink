# mobile/

Purpose

- Hosts the React Native CLI mobile app, UI components, hooks, and mobile-specific services.

Ownership

- Mobile engineering team

Allowed dependencies

- UI libs, React Native, animation libs, build tools.

Forbidden responsibilities

- Backend/API implementations, payment processing logic, persistent database design.

Future phases

- App screens, native integrations, release pipelines.

Folder boundaries

- Keep platform-specific code here. Shared logic must live in `../shared`.
